# dateTime Property Method - API Reference

This method defines a timestamp property.

## Example

```ts
import { model } from "@medusajs/utils"

const MyCustom = model.define("my_custom", {
  date_of_birth: model.dateTime(),
  // ...
})

export default MyCustom
```
