# restorePriceLists - Pricing Module Reference

This documentation provides a reference to the `restorePriceLists` method. This belongs to the Pricing Module.

This method restores a soft deleted price lists by its IDs.

## Example

```ts
await pricingModuleService.restorePriceLists([
  "plist_123",
  "plist_321",
])
```
