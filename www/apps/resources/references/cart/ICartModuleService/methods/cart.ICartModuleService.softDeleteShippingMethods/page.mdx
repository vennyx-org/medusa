# softDeleteShippingMethods - Cart Module Reference

This documentation provides a reference to the `softDeleteShippingMethods` method. This belongs to the Cart Module.

This method soft deletes shipping methods by their IDs.

## Example

```ts
await cartModuleService.softDeleteShippingMethods([
  "casm_123",
  "casm_321",
])
```
