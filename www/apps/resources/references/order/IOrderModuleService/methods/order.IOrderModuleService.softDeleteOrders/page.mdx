# softDeleteOrders - Order Module Reference

This documentation provides a reference to the `softDeleteOrders` method. This belongs to the Order Module.

This method soft deletes orders by their IDs.

## Example

```ts
await orderModuleService.softDeleteOrders(["123", "321"])
```
