# createOrderAddresses - Order Module Reference

This documentation provides a reference to the `createOrderAddresses` method. This belongs to the Order Module.

## createOrderAddresses(data, sharedContext?): Promise&#60;[OrderAddressDTO](../../../interfaces/order.OrderAddressDTO/page.mdx)[]&#62;

This method creates addresses.

### Example

```typescript
const addresses = await orderModuleService.createOrderAddresses([

    first_name: "John",

    province: "AnyState",
    postal_code: "12345"
  }
```
