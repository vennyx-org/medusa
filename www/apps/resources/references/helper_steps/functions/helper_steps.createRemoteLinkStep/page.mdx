# createRemoteLinkStep - Helper Steps API Reference

This documentation provides a reference to the `createRemoteLinkStep` step. It belongs to the `@medusajs/medusa/core-flows` package.

## Example

```ts
import { createWorkflow } from "@medusajs/workflows-sdk"
import { createRemoteLinkStep } from "@medusajs/core-flows"
import { Modules } from "@medusajs/utils"

const helloWorldWorkflow = createWorkflow("hello-world", () => {
  createRemoteLinkStep([
    {
      [Modules.PRODUCT]: {
        product_id: "prod_123",
      },
      helloModuleService: {
        my_custom_id: "mc_123",
      },
    },
  ])
})
```
