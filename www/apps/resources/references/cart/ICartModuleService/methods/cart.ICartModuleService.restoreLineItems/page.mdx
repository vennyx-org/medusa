# restoreLineItems - Cart Module Reference

This documentation provides a reference to the `restoreLineItems` method. This belongs to the Cart Module.

This method restores soft deleted line items by their IDs.

## Example

```ts
await cartModuleService.restoreLineItems([
  "cali_123",
  "cali_321",
])
```
