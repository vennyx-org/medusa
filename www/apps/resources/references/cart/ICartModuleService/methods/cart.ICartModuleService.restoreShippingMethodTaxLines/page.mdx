# restoreShippingMethodTaxLines - Cart Module Reference

This documentation provides a reference to the `restoreShippingMethodTaxLines` method. This belongs to the Cart Module.

This method restores soft deleted shipping method tax lines by their IDs.

## Example

```ts
await cartModuleService.restoreShippingMethodTaxLines([
  "casmtxl_123",
  "casmtxl_321",
])
```
