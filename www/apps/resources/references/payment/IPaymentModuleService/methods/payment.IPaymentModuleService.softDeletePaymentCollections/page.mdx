# softDeletePaymentCollections - Payment Module Reference

This documentation provides a reference to the `softDeletePaymentCollections` method. This belongs to the Payment Module.

This method soft deletes payment collections by their IDs.

## Example

```ts
await paymentModule.softDeletePaymentCollections(["paycol_123"])
```
