# restoreOrderLineItemTaxLines - Order Module Reference

This documentation provides a reference to the `restoreOrderLineItemTaxLines` method. This belongs to the Order Module.

This method restores soft-deleted line item tax lines by their IDs.

## Example

```ts
await orderModuleService.restoreOrderLineItemTaxLines([
  "123", "321"
])
```
