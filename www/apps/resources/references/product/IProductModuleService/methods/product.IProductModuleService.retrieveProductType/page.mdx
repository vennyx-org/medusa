# retrieveProductType - Product Module Reference

This documentation provides a reference to the `retrieveProductType` method. This belongs to the Product Module.

This method is used to retrieve a product type by its ID.

## Example

```ts
const productType =
  await productModuleService.retrieveProductType("ptyp_123")
```
