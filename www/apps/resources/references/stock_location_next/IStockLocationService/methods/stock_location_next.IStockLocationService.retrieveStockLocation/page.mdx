# retrieveStockLocation - Stock Location Next Module Reference

This documentation provides a reference to the `retrieveStockLocation` method. This belongs to the Stock Location Next Module.

This method retrieves a stock location by its ID.

## Example

```ts
const stockLocation =
  await stockLocationModuleService.retrieveStockLocation("sloc_123")
```
