# createCustomerAddresses - Customer Module Reference

This documentation provides a reference to the `createCustomerAddresses` method. This belongs to the Customer Module.

## createCustomerAddresses(addresses, sharedContext?): Promise&#60;[CustomerAddressDTO](../../../interfaces/customer.CustomerAddressDTO/page.mdx)[]&#62;

This method adds addresses to customers.

### Example

```ts
const addresses = await customerModuleService.createCustomerAddresses([
  {
    customer_id: "cus_123",
    address_name: "Home",
    address_1: "432 Stierlin Rd",
    city: "Mountain View",
    country_code: "us",
  },
])
```
