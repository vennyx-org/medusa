# setOrderShippingMethodTaxLines - Order Module Reference

This documentation provides a reference to the `setOrderShippingMethodTaxLines` method. This belongs to the Order Module.

This method set the shipping method tax lines of an order.

## Example

```ts
const shippingMethodTaxLines = await orderModuleService
  .setOrderShippingMethodTaxLines(
    "123",
    [
      {
        code: "123",
        rate: 2
      },
      {
        id: "321",
        rate: 2
      }
    ]
  )
```
