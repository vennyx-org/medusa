# registerShipment - Order Module Reference

This documentation provides a reference to the `registerShipment` method. This belongs to the Order Module.

This method registers a shipment for an order, return, claim, or exchange.

## Example

```ts
await orderModuleService.registerShipment({
  order_id: "123",
  items: [
    {
      id: "321",
      quantity: 1
    }
  ]
})
```
