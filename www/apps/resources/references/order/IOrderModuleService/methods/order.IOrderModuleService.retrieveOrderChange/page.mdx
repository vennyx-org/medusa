# retrieveOrderChange - Order Module Reference

This documentation provides a reference to the `retrieveOrderChange` method. This belongs to the Order Module.

This method retrieves an order change by its ID.

## Example

A simple example that retrieves an order change by its ID:

```ts
const orderChange = await orderModuleService.retrieveOrderChange(
  "123"
)
```

To specify relations that should be retrieved:

```ts
const orderChange = await orderModuleService.retrieveOrderChange(
  "123",
  {
    relations: ["order"]
  }
)
```
