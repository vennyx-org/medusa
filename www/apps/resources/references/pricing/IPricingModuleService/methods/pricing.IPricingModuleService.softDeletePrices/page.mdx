# softDeletePrices - Pricing Module Reference

This documentation provides a reference to the `softDeletePrices` method. This belongs to the Pricing Module.

This method soft deletes prices by their IDs.

## Example

```ts
await pricingModuleService.softDeletePrices([
  "price_123",
  "price_321",
])
```
