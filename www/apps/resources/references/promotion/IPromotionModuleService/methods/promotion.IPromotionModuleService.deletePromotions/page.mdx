# deletePromotions - Promotion Module Reference

This documentation provides a reference to the `deletePromotions` method. This belongs to the Promotion Module.

## deletePromotions(ids, sharedContext?): Promise&#60;void&#62;

This method deletes promotions by their IDs.

### Example

```ts
await promotionModuleService.deletePromotions([
  "promo_123",
  "promo_321",
])
```
