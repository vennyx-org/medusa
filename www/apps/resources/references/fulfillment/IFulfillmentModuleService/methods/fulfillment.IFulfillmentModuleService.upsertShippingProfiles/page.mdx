# upsertShippingProfiles - Fulfillment Module Reference

This documentation provides a reference to the `upsertShippingProfiles` method. This belongs to the Fulfillment Module.

## upsertShippingProfiles(data, sharedContext?): Promise&#60;[ShippingProfileDTO](../../../interfaces/fulfillment.ShippingProfileDTO/page.mdx)[]&#62;

This method updates existing shipping profiles, or creates new ones if they don't exist.

### Example

```ts
const productTags = await productModuleService.upsertShippingProfiles([
  {
    id: "id_1234",
    metadata: {
      test: true,
    },
  },
  {
    name: "Digital",
  },
])
```
