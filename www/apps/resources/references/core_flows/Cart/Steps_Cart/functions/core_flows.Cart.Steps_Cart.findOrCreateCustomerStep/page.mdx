# findOrCreateCustomerStep - Medusa Workflows API Reference

This documentation provides a reference to the `findOrCreateCustomerStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step either finds a customer matching the specified ID, or finds / create a customer
matching the specified email. If both ID and email are provided, ID takes precedence.
