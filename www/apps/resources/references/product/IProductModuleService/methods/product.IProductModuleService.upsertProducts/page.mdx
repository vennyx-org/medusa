# upsertProducts - Product Module Reference

This documentation provides a reference to the `upsertProducts` method. This belongs to the Product Module.

## upsertProducts(data, sharedContext?): Promise&#60;[ProductDTO](../../../interfaces/product.ProductDTO/page.mdx)[]&#62;

This method updates existing products, or creates new ones if they don't exist.

### Example

```ts
const products = await productModuleService.upsertProducts([
  {
    id: "prod_123",
    handle: "pant",
  },
  {
    title: "Shirt",
  },
])
```
